# Minimal Coffee Shop Landing Page

A modern and minimal **coffee shop landing page** built with **HTML, CSS and JavaScript**.
The project focuses on clean typography, elegant layout and smooth scroll-based animations to create a premium café website experience.

The design follows a minimal aesthetic inspired by modern Nordic coffee brands.

## Live Demo

https://patris-web.github.io/minimal-coffee-shop/

---

## ✨ Features

* Clean minimal UI design
* Fully responsive layout (desktop, tablet, mobile)
* Smooth scroll-based animations
* Image reveal animations
* Modern typography and layout
* Lightweight structure without frameworks
* Easy to customize

---

## 🧰 Technologies

* HTML5
* CSS3
* JavaScript
* GSAP (animations)

## 🚀 Getting Started

1. Download or clone the repository
2. Open the project folder
3. Launch `index.html` in your browser

No build tools or dependencies required.
